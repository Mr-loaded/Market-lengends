# Market Legends 🏆

Nigeria's first financial literacy gaming platform.

## Deploy to Netlify via GitHub

1. Upload all these files to a GitHub repository
2. Go to app.netlify.com
3. Click "Add new site" → "Import an existing project"
4. Connect your GitHub account and select this repo
5. Build settings are auto-detected from netlify.toml:
   - Build command: `npm install && npm run build`
   - Publish directory: `dist`
6. Click Deploy

Netlify will automatically build and deploy on every push to GitHub.

## Local Development

```bash
npm install
npm run dev
```

## Tech Stack
- React 18 + Vite
- Firebase Realtime Database
- Deployed on Netlify
